Tentang VPS Ini

Bila mendengar kata VPS (Virtual Private Server), biasanya kita terbayang pada suatu server yang biayanya ratusan ribu rupiah perbulan. Tapi tidak untuk VPS ini. Tarif sewanya hanya $1/bulan atau hanya setara dengan satu mangkok bakso. Karena murah, tentu saja spec-nya cukup rendah. RAM hanya 128 MB dan disk hanya 20 GB. Spec sekecil ini cuma cocok untuk Slave DNS. Tapi selain difungsikan sebagai Slave DNS, server ini ternyata mampu dipasangi Nginx. Yowis lah, akhirnya saya membuat situs web ini yang dibuat sangat sederhana. Tidak memakai PHP dan MySQL karena pasti RAM yang tersedia tidak cukup.

Akhir kata, semoga situs web ini berguna bagi kita semua. Bila anda punya waktu senggang, saya mengundang anda untuk mengunjungi blog saya yaitu Mawan.net
